Roche Centralized Data Manager Platform

Web Application

ABOUT PROJECT


Challenge:

Designing a Centralized Data Manager platform for Roche in order to connect different analyzers (instruments) of the x800 family which consists of Cobas 6800/8800 and STING analyzers. The new Data Manager shall consolidate certain functionalities, act as a centralized database, act as a single point of communication to other systems and connect the instrument within the lab.



Roche

Client

8 Months

Project Duration (My Involvement)

60+ Members

Team Size



Outcome:

Roche Centralized Data Manager Platform: Data Manager is a software platform for Roche x800 device family with a web based system that manages the sample order execution & its results handling. It also provides inter connectivity with Analytic devices & Host Systems/LIS that communicates with standard protocols.
